# ProjectHub - Project Management Dashboard

A high-tech project management dashboard with daily log support for photos and videos.

## Features

- Dashboard with stats overview
- Project management (CRUD)
- Daily activity logs
- Photo support (Cloudinary URLs)
- Video support (YouTube embeds / Cloudinary)
- Dark modern UI with Tailwind CSS
- Local storage persistence
- Deployable to GitHub Pages

## Setup

```bash
npm install
npm run dev
```

## Deployment

Push to `main` branch — GitHub Actions automatically deploys to GitHub Pages.

## Adding Media

- **Photos:** Upload to Cloudinary, paste the URL
- **Videos:** Upload to YouTube (or Cloudinary), paste the URL
